## Can I Convert HTML Strings, Files, and Live URLs?

Absolutely. IronPDF supports all three cases:

### How Do I Convert an HTML String with Dynamic Data?

Perfect for templated content like invoices or reports:

```csharp
// Install-Package IronPdf
using IronPdf;

string html = $@"
  <h2>Invoice #{4567}</h2>
  <p>Date: {DateTime.Today:yyyy-MM-dd}</p>
";
var renderer = new ChromePdfRenderer();
var pdf = renderer.RenderHtmlAsPdf(html);
pdf.SaveAs("dynamic-invoice.pdf");
```

String interpolation works well for injecting data.

### How Do I Convert a Local HTML File with Relative Assets?

If your HTML references local CSS, JS, or images, set the base path:

```csharp
// Install-Package IronPdf
using IronPdf;

var renderer = new ChromePdfRenderer();
renderer.RenderingOptions.LocalFileRootPath = @"C:\Reports\assets";

var pdf = renderer.RenderHtmlFileAsPdf(@"C:\Reports\monthly.html");
pdf.SaveAs("monthly-report.pdf");
```

See [Base Url Html To Pdf Csharp](base-url-html-to-pdf-csharp.md) for deeper tips on asset paths.

### How Do I Convert a Live Web Page (Including JavaScript)?

For client-side rendered pages or dashboards:

```csharp
// Install-Package IronPdf
using IronPdf;

var renderer = new ChromePdfRenderer();
renderer.RenderingOptions.EnableJavaScript = true;
renderer.RenderingOptions.WaitFor.RenderDelay = 2500; // milliseconds

var pdf = renderer.RenderUrlAsPdf("https://your-app-site.com/report");
pdf.SaveAs("web-report.pdf");
```

You can also wait for specific elements instead of using a fixed delay.

## What Are Some Common HTML-to-PDF Pitfalls and How Do I Fix Them?

- **CSS/JS Not Loading:** Use absolute paths or configure `LocalFileRootPath`.
- **Images or Fonts Missing:** Double-check asset URLs and permissions, or see [Web Fonts Icons Pdf Csharp](web-fonts-icons-pdf-csharp.md).
- **Dynamic Content Not Rendering:** Enable JavaScript and add a render delay.
- **PDF Looks Different Than Browser:** Set the CSS media type and adjust margins.
- **Protected Pages:** Provide authentication cookies or export HTML after login.

For more troubleshooting, see the [Convert Html To Pdf Csharp](convert-html-to-pdf-csharp.md) FAQ.
